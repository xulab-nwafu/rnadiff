About
-----
These scripts are part of `hits` pipeline for RNAseq, but also work independently.


rnadiff
---------------
```
Usage: rnadiff rnaseq.conf rnaseq.tsv control_tag case_tag
 e.g.: rnadiff demo/rnaseq.conf demo/rnaseq.tsv wt mu
```
